# E-Commerce Website

## 📌 Description
A full-stack e-commerce web application for browsing and purchasing products with integrated front-end and back-end functionality.

## 🚀 Features
- Product listing and browsing
- User interaction and navigation
- Backend data handling
- Responsive UI design

## 🛠️ Tech Stack
HTML, CSS, JavaScript, PHP, MySQL
